# 형상/시험 변수 기반 고압차단기 성능 예측 모델 분석 보고서

## 1. 요약
본 문서는 '형상/시험 변수 기반 고압차단기 성능 예측' 분류 모델의 학습 결과 및 후보 모델 비교에 대한 분석 결과를 제공합니다. 현재 선택된 모델은 Gradient Boosting Classifier이며, 검증 데이터(30개 행)를 기준으로 평가되었습니다. 전반적인 분류 성능 지표는 중간 수준으로 나타났으며, 특히 F1 점수가 낮아 추가적인 성능 개선 또는 모델 재검토가 필요함을 시사합니다.

## 2. 데이터 및 검증 조건
1. **데이터셋 정보:** 분석 대상 데이터셋은 총 293개의 행을 포함하고 있으며, 특징(Feature) 수는 56개입니다.
2. **모델 스키마:** 타겟 변수(`label`)를 예측하는 분류 작업으로 설정되었으며, 모델 학습에 사용된 주요 입력 변수는 `Result`, `TRVmax[kV]`, `Gas type`, `Gas_P[bar.g]`, `Tarc[ms]` 등 56개의 특징을 활용했습니다.
3. **검증 조건:** 모델은 훈련 데이터의 90%를 학습에 사용하고 나머지 10%를 검증 세트로 사용하여 평가되었습니다 (Validation Rows: 30).

## 3. 모델 성능 평가
선택된 최적 모델인 Gradient Boosting Classifier의 검증 결과는 다음과 같습니다.

*   **Accuracy (정확도):** 0.5667
*   **AUC (Area Under the Curve):** 0.64
*   **Recall (재현율):** 0.5333
*   **Precision (정밀도):** 0.5714
*   **F1 Score:** 0.5517
*   **MCC (Matthews Correlation Coefficient):** 0.1336

## 4. 후보 모델 비교
다양한 분류 알고리즘을 비교한 결과, Gradient Boosting Classifier가 현재 선택된 모델이지만 다른 후보 모델들과의 성능 차이를 고려할 필요가 있습니다.

| 모델명 | Accuracy | AUC | Recall | Precision | F1 | MCC |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Gradient Boosting Classifier (선택) | 0.5667 | 0.64 | 0.5333 | 0.5714 | 0.5517 | 0.1336 |
| Random Forest Classifier | 0.6426 | 0.6709 | 0.6538 | 0.6551 | 0.6432 | 0.297 |
| Light Gradient Boosting Machine | 0.6883 | 0.7243 | 0.6846 | 0.7043 | 0.6817 | 0.3923 |
| Ada Boost Classifier | 0.6583 | 0.6874 | 0.6154 | 0.6744 | 0.6369 | 0.3243 |

**해석 의견:**
후보 모델들 중 Random Forest 및 Light Gradient Boosting Machine은 선택된 모델보다 훨씬 높은 성능 지표(특히 F1 점수와 AUC)를 보였습니다. 이는 데이터의 복잡한 패턴을 더 잘 포착할 수 있음을 의미합니다. 따라서 최종 모델 선택 시 단순히 현재 저장된 모델뿐만 아니라, 후보 모델들의 높은 성능과 해석 가능성을 함께 고려해야 합니다.

## 5. 검증 시각화 해석
제공된 시각화 결과(Confusion Matrix, Classification Report, AUC Curve, Precision-Recall Curve 등)는 모델의 실제 분류 능력과 예측 경계를 시각적으로 보여줍니다. 특히 혼동 행렬(Confusion Matrix)은 실제 클래스(0 또는 1)와 모델이 예측한 클래스 간의 오분류 패턴을 명확히 파악하는 데 중요합니다.

## 6. 결론
### 종합 판단
- `Gradient Boosting Classifier`은 현재 저장된 검증 데이터 기준으로 선택된 모델이며, F1/AUC/Accuracy 기준 주요 성능은 0.5517입니다.
- 현재 세션에는 예측용 파일 결과가 없어 저장 모델의 검증 성능과 후보 모델 비교 결과를 중심으로 결론을 작성했습니다.

### 성능 및 모델 비교
- 후보 모델 비교 관점에서는 F1는 높을수록 유리한 지표이며, 후보 중 `Light Gradient Boosting Machine`의 값이 0.6817로 가장 우수합니다.
- 현재 선택 모델은 `Gradient Boosting Classifier`이며, 후보 비교 결과와 저장 모델 검증 지표를 함께 기준으로 판단해야 합니다.
- 주요 진단 포인트는 검증 세트는 30개 row 기준으로 평가되었습니다.

### 예측용 파일 반영
- 별도 예측용 파일 결과가 없으므로, 실제 배치 예측 분포나 row별 오차 평가는 포함하지 않았습니다.
- 다음 분석 자료에서는 예측용 Excel 결과를 함께 넣으면 모델 검증 결과와 실제 적용 데이터의 분포 차이를 같이 판단할 수 있습니다.

### 운영 적용 시 유의사항
- 결론적으로 본 모델은 현재 데이터 범위 내 예측 지원에 사용할 수 있으나, 검증 세트는 30개 row 기준으로 평가되었습니다.
- 신규 설계 조건이나 기존 데이터 범위를 벗어난 입력에 대해서는 예측 신뢰도가 낮아질 수 있으므로 별도 검증이 필요합니다.

### 후속 작업
- 신규 시험 데이터 또는 별도 holdout set으로 재검증하고, 오차가 큰 row의 입력 변수 조건을 우선 점검해야 합니다.
- 후보 모델 간 성능 차이가 작다면 성능 지표뿐 아니라 해석성, 추론 속도, 재현성을 함께 기준으로 최종 운용 모델을 선택해야 합니다.
